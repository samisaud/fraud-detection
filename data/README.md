# Data Directory

## Raw Data

The raw dataset (`data/raw/creditcard.csv`) is **not committed to this repository**.
It is versioned separately via DVC.

### Download Instructions

1. Go to: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
2. Click **Download** (requires free Kaggle account)
3. Extract `creditcard.csv` to `data/raw/creditcard.csv`

### Dataset Info

| Property | Value |
|----------|-------|
| Source | ULB Machine Learning Group |
| Licence | CC0 Public Domain |
| Rows | 284,807 transactions |
| Features | 30 (V1–V28 PCA anonymised + Time + Amount) |
| Target | `is_fraud` (0 = legitimate, 1 = fraud) |
| Fraud rate | 0.172% (492 fraud cases) |
| Size | ~144 MB |

### Why this dataset?

- **Public domain (CC0)** — safe to use in public repos, no attribution required
- **Realistic class imbalance** — mirrors production fraud detection scenarios
- **Industry-standard benchmark** — used in hundreds of peer-reviewed papers
- **Anonymised** — no PII, fully safe to share

## Processed Data

Files in `data/processed/` are generated by `dvc repro` and tracked by DVC.
They are not committed to Git.

| File | Stage | Description |
|------|-------|-------------|
| `train.csv` | prepare | Training split (80%) |
| `test.csv` | prepare | Test split (20%) |
| `train_features.csv` | featurize | Scaled + resampled training features |
| `test_features.csv` | featurize | Scaled test features |
